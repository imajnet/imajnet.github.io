Scanning

Production scanning is done by high-speed, high-volume KODAK ® scanners and PSICapture ® software, which automatically indexes documents during the scan process. Using PSICapture ® the scanner can lift data from right off the page to index, claims forms or invoices. This highly accurate software automates workflow, extracting data from documents for use in other applications.
